# UIKit기반 앱의 간단한 화면 인터페이스 구조


UIKit앱의 화면을 구성하는데에 있어서 중요한 요소는 아래와 같다.  

- UIScreen : 기기에 연결되는 물리적인 화면을 정의하는 객체
- UIWindow : 화면 그리기 지원 도구를 제공하는 객체
- UIView : 그리기를 수행할 객체 키트
