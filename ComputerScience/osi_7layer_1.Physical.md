# OSI 7 Layer - 1. Physical Layer(물리계층)

물리적으로 연결된 두 대의 컴퓨터가 0과 1의 나열을 주고 받을 수 있게 해주는 모듈이다.

## Physical Layer의 기능

- Encoding : 0과 1의 나열을 아날로그 신호로 전선으로 흘려보낸다.
- Decoding : 전선을 통해 아날로그 신호가 들어오면 0과 1의 나열로 해석한다.

## 세부동작

### Encoding

- Encoding : 데이터를 보내는 컴퓨터 -> 1계층의 Encoder
  - 0과 1로 이루어진 데이터를 1계층 Encode의 Input에 넣어준다.
  - 1계층 Encoder에서 데이터를 아날로그 신호로 전환한다.
