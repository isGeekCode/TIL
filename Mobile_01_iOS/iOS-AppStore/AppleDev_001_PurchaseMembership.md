
# Apple 개발자 계정 구매 절차

Apple 개발자 계정(Apple Developer Program Membership)은 iOS, macOS, watchOS, tvOS 앱을 배포하거나 각종 Apple의 개발 서비스(예: TestFlight, 푸시 알림 등)를 사용하기 위해 필수입니다. 아래 단계별로 Apple 개발자 계정 구매 과정을 정리하였습니다.

> ⚠️ 본 글은 개인 개발자 계정 등록 과정을 다룹니다. 조직 계정(Organization) 전환 과정은 별도 글에서 확인하세요.


## 단계 1. Apple ID 준비

Apple 개발자 프로그램에 가입하려면 Apple ID가 필요합니다. 기존 Apple ID가 있다면 사용 가능하며, 없다면 [Apple ID 생성 페이지](https://appleid.apple.com/)에서 새로 만드세요. Apple ID의 이메일 인증, 2단계 인증(이중 인증) 설정이 필요합니다.

## 단계 2. 개발자 프로그램 가입 페이지 접속

[Apple Developer Program 가입 페이지](https://developer.apple.com/programs/)에 접속합니다. "Enroll" 또는 "가입하기" 버튼을 클릭하여 절차를 시작합니다.

## 단계 3. 약관 동의 및 정보 입력

Apple ID로 로그인 후, 약관에 동의합니다. 개인 개발자 계정과 회사(법인) 계정 중 선택이 필요하며, 회사 계정은 D-U-N-S 번호 등 추가 정보가 필요합니다. 이름, 주소, 연락처 등 기본 정보를 입력합니다.
여기선 개인 개발자로 진행

## 단계 4. 결제 진행

프로그램 연회비(USD 99, 한화 약 129,000원)를 결제해야 합니다. 신용카드, 일부 국가는 Paypal 등 결제 수단을 지원합니다. 결제 정보를 입력하고 결제를 완료합니다.

## 단계 5. 이메일 승인 및 계정 활성화

결제 후, Apple에서 확인 이메일을 발송합니다. 확인 절차를 완료하면 계정이 활성화됩니다. 일반적으로 1~2일 내에 완료됩니다.

## 신청 완료

이제 Apple 개발자 계정이 활성화되었습니다. [Apple Developer 사이트](https://developer.apple.com/)에서 개발자 서비스 이용이 가능하며, Xcode를 통한 앱 배포, TestFlight 베타 테스트, 인증서·프로비저닝 프로파일 발급 등이 가능합니다.
